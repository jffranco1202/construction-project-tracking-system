## 📌 Construction Project Tracking System

### 📖 Overview
This project is a structured Excel-based system designed to track construction progress and performance.

---

### 🎯 Purpose
- Monitor project progress
- Improve reporting accuracy
- Support decision-making

---

### 🛠 Tools Used
- Microsoft Excel
- MS Project (optional)

---

### ⚙️ Features
- Automated progress computation
- Dashboard visualization
- Weighted percentage tracking
